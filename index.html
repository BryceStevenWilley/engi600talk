Feasible Motion Planning

  • Finds a connected path from start to goal in the configuration space
  • Doesn't collide with any obstacles

Optimal Motion Planning

  • Assign a cost to every path
  • Find a feasible path that minimized the cost

Pros

  • If a solution exists, it will be found
  • Conceptually simple
  • Generalizable
  • Optimal planners exist

Cons

  • Can't conclusively say "there is no solution"
  • Must smooth the path after finding it
  • Optimal planners converge very slowly

Pros

  • Fast! (5-10x by our experiments)
  • Quality paths

Cons

  • Paths aren't globally optimal
  • Robot specific
  • No guarantee of finding a feasible path
Surprisingly, there haven't been any good comparisons in the literature.
